# TinyDB 架构设计文档

## 1. 概述

TinyDB 是一个轻量级 C/S 架构关系型数据库系统，采用 C++17 开发，旨在通过渐进式开发方式，逐步构建完整的数据库功能。

## 2. 技术栈

- **编程语言**: C++17
- **构建工具**: CMake 3.14+
- **SQL 解析**: Flex + Bison
- **测试框架**: Google Test
- **网络协议**: 自定义二进制协议
- **端口**: 5432

## 4. 模块设计

### 4.1 网络层 (network/)

- **Server**: TCP 服务器，多线程处理客户端连接
- **Connection**: 连接管理，处理单个客户端会话
- **Protocol**: 自定义二进制协议定义

### 4.2 SQL 解析层 (sql/)

- **Lexer (Flex)**: 词法分析，生成 Token 序列
- **Parser (Bison)**: 语法分析，构建语法树
- **Token**: Token 类型定义

### 4.3 执行层 (engine/)

- **Executor**: 简单执行器，阶段一仅返回预定义响应

### 4.4 公共组件 (common/)

- **Types**: 基础类型定义
- **Error**: 错误码体系
- **Logger**: 日志系统

## 5. 通信协议设计

### 5.1 消息格式

```
+--------+--------+--------+--------+--------+
|  Magic |Version |  Type  |  Flags |Reserved|
| 1 byte | 1 byte | 1 byte | 1 byte | 4 bytes|
+--------+--------+--------+--------+--------+
|                Body Length                 |
|                 4 bytes                    |
+-------------------------------------------+
|                   Body                     |
|                 Variable                   |
+-------------------------------------------+
```

- **Magic**: 0x54 ('T')
- **Version**: 0x01
- **Type**: 消息类型 (SQL命令、响应、心跳等)
- **Body Length**: Body 长度（大端序）

### 5.2 消息类型

| 类型 | 值 | 说明 |
|------|-----|------|
| SQL_COMMAND | 0x01 | SQL 命令 |
| SQL_RESPONSE | 0x02 | SQL 响应 |
| HEARTBEAT | 0x03 | 心跳检测 |
| ERROR | 0x04 | 错误响应 |

## 8. 验收标准

```bash
# 启动服务器
./dbserver -p 5432

# 客户端连接并执行 SQL
./dbclient -h localhost -p 5432 "SELECT 1"

# 交互式客户端
./tinydb-cli -h localhost -p 5432

# 执行测试
./tests/test_phase5_integration
```
